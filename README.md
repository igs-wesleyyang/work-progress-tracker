# 市場組工作進度追蹤

整併美國企場 / 亞洲企劃 / Arthur 三張來源表（唯讀），呈現一頁式的全組工作進度看板。

## 功能
- 依團隊 → 成員分組，列出每人當週工作項目與預計工時
- 工作週期：每週三 ~ 隔週二（自動計算本週區間）
- 每項可更新「完成度 %」，狀態自動變色、記錄最後更新時間
- 可依成員篩選、展開/收合、匯出 / 匯入 JSON

## 使用
直接開啟 `index.html` 即可（純前端，無外部相依）。

## 待辦（第二階段）
- 多人即時同步進度（需後端 / 雲端資料庫）
- 存取登入保護（限公司同仁）
- Telegram 定時提醒（週一二 20:00、週三四五 17:30，台灣時間）

> 進度與工時目前儲存在「本機瀏覽器」，尚未跨裝置同步。
